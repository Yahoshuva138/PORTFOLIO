# Personal Portfolio

A simple responsive portfolio website built with HTML, CSS, and JavaScript.

## Files
- index.html - Main page structure
- style.css - Visual styling and responsive layout
- script.js - Mobile navigation and footer year update
- images/profile.svg - Placeholder profile image
- assets/resume.pdf - Placeholder resume download

## Customize
- Replace the placeholder content in index.html with your own bio, projects, and contact details.
- Update the profile image and resume files in the images/ and assets/ folders.
- Open index.html in a browser to preview the site.
